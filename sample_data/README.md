# Sample Data Information

This directory contains example sales data for testing the AI Sales Analyzer.

## Files

### example_sales.xlsx
Real sample data from a skincare e-commerce business (Bloom Co).

**Columns**:
- Date: Order date
- Order_ID: Order number
- Product_Name: Product name
- Quantity: Items sold
- Price: Unit price (USD)
- State: Customer state
- Total: Order total (USD)

**Data Range**: November 18-24, 2024  
**Records**: ~50 orders

## Usage

1. **Test the Application**:
   - Upload this file to test all features
   - Verify charts and metrics
   - Test AI assistant with real data

2. **Format Reference**:
   - Use as a template for your own data
   - Shows correct column structure
   - Demonstrates data quality

## Note

This is real anonymized sales data. The application automatically maps column names like `Order_ID` to the standard `Order ID` format.
